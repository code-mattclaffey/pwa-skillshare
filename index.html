PWA Kata

An introduction in building your first PWA web app. Shall we start?

Step One - Build a manifest.json

A manifest is the first thing you need to do when building a PWA. When the user adds your website to the homescreen of their device, the manifest communicates to the device telling them how the website should display & what icon should be shown to the user on the homescreen/splash page.
